
# Adit's Social Button Generator

This is **Adit's Social Button Generator** simple WebApp that was created for the purpose of decorating github README profile, this app can create social media buttons like these and you can customize the text and the color of button aswell, primarily created for decorating github README, this social media button can be used for others though, if you wanted too

To start making your own social media button for whatever your purpose is, just visit: [adit.madbyad.my.id/adit-social-button-generator](https://adit.madbyad.my.id/adit-social-button-generator)
